# Pensamento-Computacional

Inicialmente foi montado toda a estrutura composta por um arduino, uma placa de ensaio, fios, três resitores, três potênciometros e um led rgb para que houvesse a funcionalidade solicitada.
Em seguida foi criada as variáveis "pote_A2", "pote_A3" e "pote_A4".
Posteriormente a variáveis foram definidas como pino analógicos.
Após essa sequência, quando acionamos o potênciometro 1 ele ativa a cor vermelha do lgb, quando acionamos o potênciometro 2 ele ativa a cor azul do lgb e por fim acionamos o potênciometro 3 ele ativa a cor verde lgb, entregando assim funcionalidade inicialmente solicitada.
